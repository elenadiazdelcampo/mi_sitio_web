Proyectos académicos

Extración de ficheros de audio a partir de la URL de clips de YouTube

Herramientas utilizadas: IceGrid, Python

El siguiente proyecto fue realizado en las prácticas de la asignatura de Sistemas Distribuidos, cuyo objetivo es diseñar un sistema cliente-servidor que permita la extracción de ficheros de audio a partir de la URL de clips de YouTube.

Aplicación móvil para la gestión y listado de películas

Herramientas utilizadas: Android Studio, Java, SQLite Browser

Aplicación móvil básica para el sistema Android realizada en los seminarios de la asignatura Iteracción Persona Ordenador II, en la cuál se realiza la gestión y listado de películas, haciendo uso de una Base de Datos local y Android Studio.

Herramientas utilizadas: Eclipse, Java

Reconocimiento musical de canciones, similar a Shazam, usando aplicaciones como spotify, en base a la huella digital acústica implementado en Java. El código es completado respecto al material proporcionado en clase.

Diseño de la red del Campus de Ciudad Real

Herramientas utilizadas: Packet Tracer

Diseño de la red del Campus de Ciudad Real, realizado con la herramienta Packet Tracer. Dónde se ha analizan las distintas metas de negocio que me han permitido realizar tanto el diseño físico y lógico de la red de este campus.

Construcción de ruta óptimas para recorrer distintos lugares de un municipio

Herramientas utilizadas: NetBeans, Java, archivos .xml, manejo de json

El siguiente proyecto fue realizado en las prácticas de Sistemas Inteligentes. Consiste en la construcción de rutas óptimas para recorrer un conjunto de lugares de un municipio. Dónde cada municipio será tratado como un grafo, con sus nodos y aristas.
